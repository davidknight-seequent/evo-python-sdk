get_available_units(context: IContext) -> list[UnitInfo]

Get the list of available units from the Block Model Service.

Parameters:

Name Type Description Default
context IContext

The context containing environment and connector.

 required

Returns:

Type Description
list[UnitInfo]

List of available units. Example: units = await get_available_units(manager) for unit in units: print(f"{unit.unit_id}: {unit.description} ({unit.symbol})")
